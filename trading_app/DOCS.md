# Trading App — Home Assistant Add-on

Swing-Trading System mit Pattern-Erkennung.

## Installation

1. In HA: Settings → Add-ons → Add-on Store → Repositories
2. Repository hinzufügen
3. Add-on "Trading App" installieren und starten
4. In der HA Sidebar erscheint "Trading"

## Daten

- Die DB wird persistent gespeichert (überlebt Updates)
- Backups über HA Snapshots
